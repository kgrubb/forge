---------------------------
| INSTALL & UPGRADE STEPS |
---------------------------

1. Right click on the "install" file and select "Run with PowerShell". Click "ok" when prompted.

2. Start the minecraft launcher, select the forge version and click play.

3. Go to the main menu, click on "Multiplayer" -> "Add server" and add "mc.keligrubb.com:8000".

4. Connect to the server from the Multiplayer menu and have a good time!
